# Privacy Policy for Financial Estate (LinkedIn Integration)
**Last Updated: April 7, 2026**

## 1. Overview
This application ("Financial Estate Automation") is a private tool developed by Gan Chye Chuan ("Developer") for personal branding and financial advisory marketing. It uses the LinkedIn API to automate post creation and performance tracking.

## 2. Data Collection and Usage
- **Scope:** This app only accesses the LinkedIn profile data of the Developer (Gan Chye Chuan). 
- **AI Processing:** This app uses Anthropic's Claude AI to generate text and visual content for LinkedIn posts.
- **No Third-Party Sharing:** No data collected from LinkedIn (such as post engagement metrics) is sold, traded, or shared with third-party entities, except for the processing required by Anthropic's API to generate content.

## 3. 2026 Developer AI Policy Compliance
In accordance with the 2026 LinkedIn Developer AI Policy:
- **Human-in-the-Loop:** No content is published to LinkedIn without a final manual review/approval stage by the Developer.
- **AI Labeling:** All AI-generated posts include a clear disclosure or are manually verified for accuracy before distribution.
- **No Training:** Data retrieved from LinkedIn (comments/reactions) is used solely for performance analysis and is NOT used to train or fine-tune generative AI models.

## 4. Data Security
All API keys and access tokens are stored securely using GitHub Secrets and are never hard-coded or exposed publicly.
